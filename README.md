# Grocery Intelligence Website v2.3

Complete GitHub Pages replacement package.

## Improvements in this release
- The exact weekly PDF circulars are stored locally and linked from the dashboard.
- All Deals is now the default view.
- Search produces an intelligence summary with store counts, lowest advertised price and lowest reliable unit price.
- Results can be sorted by best value, price, unit price, store or brand.
- Results can show every matching deal, one best deal per store, or grouped product types.
- Giant is no longer singled out in the dashboard statistics.
- The weak overall-store winner banner and internal production language were removed.
- Exact comparison labels now require known size/count information.
- Multi-count products are compared per item when count information can be read.
- STOCK UP support remains ready but will not activate until enough historical weeks exist.

## Install
Extract this ZIP, replace the files in the local GitHub repository while preserving the hidden `.git` folder, commit, and push.
